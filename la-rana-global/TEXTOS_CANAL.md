# TEXTOS DEL CANAL — Sistema "La Rana Global"
## Plantillas de copy. Beben de la plantilla de receta. Una versión por mercado (ES / FR).

---

## 1. TÍTULO (YouTube / por receta)

Fórmula: `[Plato] + [gancho de cocina/país] + [beneficio o curiosidad]`
- Largo (YouTube): ≤60 caracteres. Ej: "Ancas de rana a la provenzal — el secreto francés"
- Corto (Reels/TikTok/Shorts): ≤40 car. Ej: "El truco francés para las ancas perfectas 🐸"
- Regla: la palabra "ancas de rana" SIEMPRE en el título (océano azul → SEO te encuentra).

## 2. DESCRIPCIÓN (YouTube)

```
[1 frase gancho de la receta].

🍽 [Nombre receta] · [país/cocina] · [tiempo] · [comensales]
🥗 Score nutricional [A-E] · [perfil saludable]
🍷 Marida con [maridaje]
📍 Ideal para [momento]

👉 Compra ancas de rana frescas: [web del mercado + UTM]

00:00 Intro
00:0X Las ancas
00:0X Técnica
00:0X El truco
00:0X Emplatado

#ancasderana #recetas [+hashtags receta]
```

## 3. CAPTION (Reels / TikTok / Shorts)

```
[Gancho 1 línea] 🐸
El truco: [truco en 1 frase].
👉 [web + UTM] (link en bio)
#ancasderana #recetafrancesa #[receta] #cocina
```

## 4. HASHTAGS BASE (siempre) + variables

- Fijos: `#ancasderana #frogslegs #grenoucerie #recetas`
- Variables por receta: país, cocina, técnica, momento _(ej. #provenzal #tapeo)_
- FR (grenoucerie.fr): `#cuissesdegrenouille #recettefrançaise`

## 5. CTA POR MERCADO (con tracking)

| Mercado | Web | UTM ejemplo |
|---|---|---|
| España | ancasderana.com | `?utm_source=youtube&utm_medium=video&utm_campaign=[receta_id]` |
| Francia | grenoucerie.fr | `?utm_source=youtube&utm_medium=video&utm_campaign=[receta_id]` |

> El UTM por `receta_id` permite saber qué vídeo lleva tráfico/ventas. Métrica clave del funnel.

## 6. MINIATURA (thumbnail) — texto en imagen

- ≤4 palabras, gran tamaño. Ej: "ANCAS A LA PROVENZAL"
- Plato emplatado + Fabien al lado. Color marca (lavanda).
